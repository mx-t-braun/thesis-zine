# Thesis Zine (StPageFlip)

This starter is 100% free/open-source and uses the StPageFlip library over a CDN.
- Live library docs: https://nodlik.github.io/StPageFlip/
- CDN used in index.html: https://unpkg.com/page-flip@2.0.7/dist/js/page-flip.browser.js

## How to use
1) Edit the `.page` blocks in `index.html` to add your content (text, images, video, audio).
2) Add new pages by duplicating a `.page` block.
3) Commit to GitHub and enable GitHub Pages (branch: main, folder: /root).
4) Link to https://YOUR-USERNAME.github.io/YOUR-REPO/

Tip: To embed YouTube/Vimeo/SoundCloud, paste their iframe/embed codes inside a `.page`.
